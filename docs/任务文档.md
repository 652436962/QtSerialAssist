# QtSerialAssist 串口调试助手 - 任务文档

## 项目概述
基于 Qt 开发的串口调试助手，支持串口通信、数据收发、十六进制/ASCII 显示、定时发送等功能。

## 功能特性

### 1. 连接方式
- [x] **串口设置**
  - [x] 串口选择（自动检测可用串口，2秒刷新）
  - [x] 波特率设置（9600 ~ 921600）
  - [x] 数据位配置（5, 6, 7, 8 位）
  - [x] 校验位设置（None, Odd, Even, Mark, Space）
  - [x] 停止位配置（1, 1.5, 2 位）
  - [x] 流控选项（None, Hardware, Software）
- [x] **网络设置**
  - [x] TCP 服务器模式
  - [x] TCP 客户端模式
  - [x] UDP 模式
  - [x] IP 地址与端口配置
  - [x] 连接状态显示

### 3. 数据发送
- [x] 文本模式发送（ASCII/HEX）
- [x] 十六进制模式发送
- [x] 定时循环发送（可配置间隔 100~60000ms）
- [x] 发送新行（支持 \r\n / \n / \r / 无）
- [x] 编码选择（UTF-8 / GBK）
- [x] 发送数据统计（字节数）

### 4. 数据接收
- [x] 文本/十六进制显示
- [x] 带时间戳显示 [hh:mm:ss.zzz]
- [x] 接收数据统计（字节数 + 帧数）
- [x] 发送数据回显（绿色 #4ecca3）
- [x] 接收数据显示（蓝色 #6eb5ff）

### 5. 控制功能
- [x] 打开/关闭连接按钮（串口+网络统一处理）
- [x] 连接状态指示（状态栏实时显示）
- [x] 清除发送/接收缓冲区
- [x] 深色/浅色主题切换
- [x] 中/英文语言切换
- [x] 设置持久化（QSettings）

### 6. 工具栏功能
- [x] 快捷指令（JSON配置化，分组管理，批量延时发送）
- [x] Modbus 指令（RTU/ASCII/TCP，CRC/LRC校验）
- [x] 终端模式（隐藏发送区，接收区可编辑，直接输入输出）
- [x] 设置（主题切换、语言切换、持久化）
- [ ] 批量发送
- [ ] 历史发送
- [ ] 自动应答
- [ ] 数据波形
- [ ] ASCII码表
- [ ] 更多工具

### 7. CI/CD
- [x] GitHub Actions 跨平台构建（Linux/Windows/macOS）
- [x] Linux AppImage / deb / rpm 打包
- [x] Windows windeployqt 自动 DLL 捆绑打包
- [x] 推送 v* 标签自动创建 GitHub Release

### 8. 国际化
- [x] 中/英文切换（QTranslator + retranslateUi）
- [x] lrelease 自动编译 .qm

- **框架**: Qt5/Qt6
- **串口库**: QSerialPort
- **构建系统**: CMake
- **UI 设计**: Qt Designer (.ui 文件)
