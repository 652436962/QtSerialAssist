# QtSerialAssist 串口调试助手 - 任务文档

## 项目概述
基于 Qt 开发的串口调试助手，支持串口通信、数据收发、十六进制/ASCII 显示、定时发送等功能。

## 功能特性

### 1. 连接方式
- [ ] **串口设置**
  - [ ] 串口选择（自动检测可用串口）
  - [ ] 波特率设置（9600, 115200 等常用速率）
  - [ ] 数据位配置（5, 6, 7, 8 位）
  - [ ] 校验位设置（None, Odd, Even, Mark, Space）
  - [ ] 停止位配置（1, 1.5, 2 位）
  - [ ] 流控选项（None, Hardware, Software）
- [ ] **网络设置**
  - [ ] TCP 服务器模式
  - [ ] TCP 客户端模式
  - [ ] UDP 模式
  - [ ] IP 地址与端口配置
  - [ ] 连接状态显示

### 3. 数据发送
- [ ] 文本模式发送
- [ ] 十六进制模式发送
- [ ] 定时循环发送（可配置间隔）
- [ ] 发送数据统计（字节数）

### 4. 数据接收
- [ ] 文本模式显示
- [ ] 十六进制模式显示
- [ ] ASCII 码显示
- [ ] 接收数据统计（字节数）
- [ ] 自动换行显示

### 5. 控制功能
- [ ] 打开/关闭连接按钮
- [ ] 连接状态指示
- [ ] 清除发送/接收缓冲区

### 6. 工具栏功能（基于工具栏截图）
-快捷指令
-批量发送
-历史发送
-自动应答
-数据波形
-modbus指令
-ASCII码表

### 7. 界面优化（基于截图参考）
- [ ] 优化串口配置面板布局
- [ ] 改进数据显示区域样式
- [ ] 添加数据帧统计图表
- [ ] 优化深色主题显示效果

## 技术实现
- **框架**: Qt5/Qt6
- **串口库**: QSerialPort
- **构建系统**: CMake
- **UI 设计**: Qt Designer (.ui 文件)
